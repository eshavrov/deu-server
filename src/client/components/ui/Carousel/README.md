Компонент `Carousel` карусель в моменте использует 5 элементов:

- пред-предыдущий элемент, нужен для плавного эффекта появления при прокрутке назад;
- предыдущий элемент;
- основной элемент;
- следующий элемент;
- новый элемент справа, нужен для плавного эффекта появления при прокрутке вперед.

Хук `useCarousel` контролирует что передать в компонент для отображения.
